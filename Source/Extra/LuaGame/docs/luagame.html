LuaGame Reference Manual

Version 0.1.0

Usage

Setup: 
using namespace Pxf::Extra::LuaGame;
Game *luagame = new Game("test.lua", m_Device);
luagames->Load();
m_Device is a pointer to your current Pxf::Graphics::Device instance.

Once every frame: 
luagame->Update(delta);
luagame->Render();
delta is a float delta value in seconds since last frame.

Basic script example

Every game script must have the following methods defined: 
function luagame:Init()
	self.GameIdent = "Our awesome testgame!"
	self.GameVersion = "1.0"
end

function luagame:PreLoad()

end

function luagame:Update(dt)

end

function luagame:Render()
	luagame.graphics.drawquad(10, 10, 40, 40)
end


Packages

    luagame.graphics

    Some awesome description about this package.

  • luagame.graphics.getscreensize()

    Gets the size of the device screen being rendered to.
      Returns
    • w, h - Width and height of screen in pixels.

  • luagame.graphics.drawquad(x, y, w, h)

    Draws a textured quad to the screen.
      Params
    • x, y - Position on screen
    • w, h - Width and height of quad

  • luagame.graphics.translate(dx, dy)

    Translate game coordinate system. (Only affects drawquad() at the moment.)
      Params
    • dx, dy - Delta translation.

  • luagame.graphics.rotate(angle)

    Rotates quads that will be drawn after this call. (Only affects drawquad() at the moment.)
      Params
    • angle - Rotation in radians.
